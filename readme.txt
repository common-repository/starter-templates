== Description ==
The All-In-One WordPress Design Package. This is an add-on for the <a href="https://wordpress.org/themes/starter-blog">Starter Blog</a> WordPress Theme for Starting a Blog.

Also, use our ever-expanding library of block-based templates to boost the Gutenberg editor.

All of our starting templates have been optimized for performance and will load faster than the rest of our website designs.

The demo material will make your website look like a theme preview, giving you a starting point for creating your own website.

Once installed and activated, browse and import demo sites from your WordPress backend with 4 steps:

1. Choice a site to import.
2. Install recommended plugins.
3. Import media and content.
4. Import config.

Done!

== Installation ==
1. Upload the entire `starter-templates` folder to the /wp-content/plugins/ directory.
2. Activate the plugin through the `Plugins` menu in WordPress.

You will find `Appearance -> Starter Templates` menu in your WordPress admin panel.

== Upgrade Notice ==
You can update the plugin using "automatic WordPress updater".
